### Getting Started:

Prerequisites

- npm
  ```sh
  install npm@latest -g
  ```

Installation

1. Clone the repo
   ```sh
   git clone git@github.com:CLSnapp/Frontend_Capstone.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the development server
    ```sh
    npm run dev
    ```
